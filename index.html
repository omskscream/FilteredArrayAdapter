CustomArrayAdapter

It allows to use any external condition for filtering elements.
Example: 

CustomArrayAdapter<Teacher> teachersAdapter = 
new CustomArrayAdapter<Teacher>(this,
    android.R.layout.simple_dropdown_item_1line, 
    DBFacade.getAllTeachers()) {
    @Override
    protected boolean isFilterCondition(final Teacher data, 
                                        String constraint) {
    String lastName = data.getLastName().toLowerCase();
    String firstName = data.getFirstName().toLowerCase();
